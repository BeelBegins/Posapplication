# ERPNext Online-First POS

A basic Electron + TypeScript desktop application shell for Windows.

It currently displays setup status only. It does not connect to ERPNext and contains no FBR, loyalty, SQLite, or POS billing implementation.

## Prerequisites

- Node.js 20 or later

## Run locally

```powershell
cd D:\erpnext-offline-pos
npm install
npm start
```

## Development mode

Use this while editing TypeScript files:

```powershell
npm run dev
```
